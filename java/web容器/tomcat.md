**三、Tomcat原理总结**

1. Tomcat需要main方法启动。

2. Tomcat需要监听本机上的某个端口。

3. Tomcat需要抓取此端口上来自客户端的链接并获得请求调用的方法与参数。

4. Tomcat需要根据请求调用的方法，动态地加载方法所在的类，完成累的实例化并通过该实例获得需要的方法最终将请求传入方法执行。

5. 将结果返回给客户端（jsp/html页面、json/xml字符串）
