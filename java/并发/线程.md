> ### 线程的 run() 和 start() 有什么区别？

- 调用 start() 方法是用来启动线程的，该线程启动后，会自动调用 run()；直接调用 run() 方法，无法达到启动多线程的目的，相当于主线程线性执行 Thread 对象的 run() 方法。
