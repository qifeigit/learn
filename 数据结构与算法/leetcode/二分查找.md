#### [33. 搜索旋转排序数组](https://leetcode-cn.com/problems/search-in-rotated-sorted-array/)

> ```
> 输入：nums = [4,5,6,7,0,1,2], target = 3
> 输出：-1
> ```



这道题首先要找出最小值，然后再按照普通的二分查找即可

```java
 while (left <= right){
            int mid = (left + right)/2;

            if(nums[mid] > nums[mid+1]){
                return mid+1;
            }
            if(nums[mid-1] > nums[mid]){
                return mid;
            }
            if(nums[mid] > nums[0]){
                left = mid;
            }
            if(nums[mid] < nums[0]){
                right = mid;
            }
        }
```
